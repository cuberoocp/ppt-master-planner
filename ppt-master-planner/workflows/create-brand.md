# Create Brand Workflow

> Standalone brand identity preset creation.

## Input

- Logo assets (PNG, SVG)
- Brand colors (hex values or brand guidelines)
- Typography preferences

## Output

Brand directory at `templates/brands/<id>/` containing:
- `brand.json` — color palette, fonts, logo references, voice guidelines
- Logo files

Brands apply at SKILL.md Step 3 via explicit user-provided directory path.
