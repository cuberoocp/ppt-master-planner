# Image Searcher Reference

## Search Sources

- Unsplash, Pexels, Pixabay for stock photography
- Wikimedia Commons for public domain / educational images
- Bing / Google image search for topical images

## Selection Criteria

- Minimum 1920x1080 for full-slide backgrounds
- Minimum 800x600 for content area images
- Prefer images with transparent backgrounds for icons/objects
- Respect licensing and attribution requirements
