# Templates

This directory contains all template resources for PPT Master Planner:

- `layouts/` — Page layout templates (SVG + design_spec per style)
- `brands/` — Brand identity presets (colors, fonts, logos)
- `charts/` — Visualization / chart SVG templates
- `icons/` — Icon libraries
- `design_spec_reference.md` — Design spec skeleton template
- `spec_lock_reference.md` — Execution contract skeleton
