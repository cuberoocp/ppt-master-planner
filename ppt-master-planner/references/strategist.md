# Strategist Role

## Eight Confirmations

The Strategist must present the following eight items to the user and wait for confirmation (BLOCKING):

1. **Canvas Format** — 16:9, 4:3, or custom
2. **Visual Theme** — Color palette, mood, style direction
3. **Typography** — Heading and body font choices
4. **Layout System** — Page structure, grid, margins
5. **Icon Style** — Icon family and usage rules
6. **Visualization Style** — Chart and diagram visual language
7. **Image Strategy** — Photography vs illustration, image sources
8. **Content Outline** — Slide-level title hierarchy and structure

## Design Spec Output

After confirmation, produce:
- `design_spec.md` — human-readable design specification
- `spec_lock.md` — machine-readable execution contract with all color, font, icon, image, layout, and chart values
